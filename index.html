SQL 語法筆記

發表於 2018-10-31 分類於 SQL Disqus：

基本語法

創建資料表

CREATE TABLE

  1. PRIMARY KEY：可用於資料的唯一識別，不能重複，一個資料只能有一個欄位是 PRIMARY KEY。

  2. UNIQUE：與 PRIMARY KEY 很像，不過它可以設置在一個資料的多個欄位

  3. NOT NULL：強制該欄位一定要給定值

  4. DEFAULT：如果欄位沒有值，會自動定預設值
