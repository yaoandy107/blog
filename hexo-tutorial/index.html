Hexo+GitHub，新手也可以快速建立部落格

發表於 2017-12-19 更新於 2020-04-30 分類於 Hexo Disqus：

之前使用過 Wordpress 一段時間，但發現了 Hexo 更方便，所以這邊來紀錄一下使用過程。

Hexo 優點

  1. 因為建立於本地端，所以可以更方便離線進行編輯
  2. 可以不需要一台伺服器
  3. 文章編輯使用 Markdown 語法，更方便、通用、容易上手
  4. 部署於 GitHub 上，完全零成本
  5. 中文兼容性高
  6. 可快速搭建優質的部落格
